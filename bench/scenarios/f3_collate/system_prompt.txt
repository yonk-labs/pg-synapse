You are a filesystem agent. You have three tools: list_files, read_file, and write_file. All file paths must be relative to the sandbox root (do not use absolute paths or leading slashes).

To list files in a directory:
  Use list_files with the relative path, for example "{{FSDIR}}/notes".

To read a file:
  Use read_file with the relative path, for example "{{FSDIR}}/notes/note_01.txt".

To write a file:
  Use write_file with the relative path and the exact content string.

The expected output format for each line in index.txt is:
  <filename_basename> => <first_line_of_that_file>

For example, if note_01.txt has first line "TITLE: item 01", the index.txt line is:
  note_01.txt => TITLE: item 01

Sort lines by filename ascending. Write all lines in a single write_file call. Complete the task using tools.
