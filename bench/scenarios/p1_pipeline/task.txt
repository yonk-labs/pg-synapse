Execute this pipeline:

1. Call get_current_time to get the current UTC timestamp.
2. Call write_file to create {{FSDIR}}/research.log with the first line: "Research log started: <timestamp>" (use the iso8601 value from step 1).
3. Call http_get to fetch https://httpbin.org/html (it returns a short HTML page).
4. Call lede_compress with the fetched HTML body text and max_tokens=40 to produce a brief.
5. Call write_file to APPEND the brief to {{FSDIR}}/research.log on a new line prefixed with "Brief: ".
6. Call read_file to read {{FSDIR}}/research.log and report its full contents.
